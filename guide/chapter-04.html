4.1 什么是CSS？ ​

什么是CSS？ ​

CSS（Cascading Style Sheets）是层叠样式表，用于描述HTML文档的呈现方式。想象一下：

  • HTML = 房子的框架结构（墙壁、门窗位置）
  • CSS = 房子的装修装饰（颜色、材质、家具摆放）
  • JavaScript = 房子的智能设备（灯光控制、电器开关）

* {
  margin: 0;
  padding: 0;
  box-sizing: border-box;

/* 后代选择器：选中所有后代元素 */
.container p {
  line-height: 1.6;
}

.menu > .menu-item {
  display: inline-block;
}

h2 + p {
  font-weight: bold;
}

/* 通用兄弟选择器：选中后面所有兄弟元素 */

  color: #666;

  border-top: none;

/* :last-child - 最后一个子元素 */
.list-item:last-child {
  border-bottom: none;
}

/* :nth-child(n) - 第n个子元素 */
.table-row:nth-child(odd) {
  background-color: #f9f9f9;
}

/* :not() - 非...的元素 */
button:not(.disabled) {
  cursor: pointer;
}

/* ==================== 5. 伪元素选择器 ==================== */

/* ::before - 在元素前插入内容 */
.quote::before {
  content: "\"";
  font-size: 24px;
  color: #42b983;
}

/* ::after - 在元素后插入内容 */
.clearfix::after {
  content: "";
  display: block;
  clear: both;
}

/* ::first-letter - 首字母 */
p.drop-cap::first-letter {

}

什么是盒子模型？ ​

盒子模型（Box Model）是CSS布局的基础，每个HTML元素都被看作一个矩形的盒子，包含四个部分：

┌─────────────────────────────────────────┐
│           Margin（外边距）                │  ← 元素与其他元素的距离
│   ┌─────────────────────────────────┐   │
│   │        Border（边框）             │   │  ← 元素的边框
│   │   ┌─────────────────────────┐   │   │
│   │   │    Padding（内边距）     │   │   │  ← 内容与边框的距离
│   │   │  ┌───────────────────┐  │   │   │
│   │   │  │  Content（内容）   │  │   │   │  ← 实际内容区域
│   │   │  └───────────────────┘  │   │   │
│   │   │      width × height     │   │   │
│   │   └─────────────────────────┘   │   │
│   └─────────────────────────────────┘   │
└─────────────────────────────────────────┘

/* ==================== 盒子模型完整示例 ==================== */

.box {
  /* ===== 内容区域 ===== */
  width: 300px;
  height: 200px;

  /* ===== 内边距（Padding）===== */
  /* 四个方向相同 */
  padding: 20px;

  /* 分别设置（上 右 下 左） */
  padding: 10px 15px 20px 25px;

  /* 单独设置某个方向 */
  padding-top: 10px;
  padding-right: 15px;
  padding-bottom: 20px;
  padding-left: 25px;

  /* 简写方式 */
  padding: 10px 20px;    /* 上下10px 左右20px */
  padding: 10px 15px 20px; /* 上10px 左右15px 下20px */

  /* ===== 边框（Border）===== */
  /* 简写：宽度 样式 颜色 */
  border: 2px solid #42b983;

  /* 分别设置 */
  border-width: 2px;
  border-style: solid;      /* solid/dashed/dotted/double/groove/ridge/inset/outset/none */
  border-color: #42b983;

  /* 单独设置某一边 */
  border-top: 3px solid #42b983;
  border-right: 2px dashed #666;
  border-bottom: 3px dotted #999;
  border-left: 2px double #333;

  /* 圆角 */
  border-radius: 8px;           /* 四个角相同 */
  border-radius: 10px 20px;     /* 左上+右下10px 右上+左下20px */
  border-radius: 10px 20px 30px; /* 左上10px 右上+左下20px 右下30px */
  border-radius: 10px 20px 30px 40px; /* 左上 右上 右下 左下（顺时针） */

  /* 单独设置某个角 */
  border-top-left-radius: 10px;
  border-top-right-radius: 20px;
  border-bottom-right-radius: 30px;
  border-bottom-left-radius: 40px;

  /* ===== 外边距（Margin）===== */
  /* 四个方向相同 */
  margin: 20px;

  /* 分别设置（上 右 下 左） */
  margin: 10px 15px 20px 25px;

  /* 单独设置某个方向 */
  margin-top: 10px;
  margin-right: 15px;
  margin-bottom: 20px;
  margin-left: 25px;

  /* 简写方式 */
  margin: 10px 20px;    /* 上下10px 左右20px */
  margin: 10px 15px 20px; /* 上10px 左右15px 下20px */

  /* 水平居中 */
  margin: 0 auto;       /* 上下0 左右自动（居中） */

  /* 负值外边距（重叠效果） */
  margin: -10px;
}

/* ==================== box-sizing属性 ==================== */

/* 模式一：content-box（标准盒模型，默认值） */
.content-box {

  width: 300px;           /* 内容宽度 */
  padding: 20px;          /* 内边距 */

  /* 内容宽度 = width - padding-left - padding-right - border-left - border-right */
  /* = 300 - 20 - 20 - 5 - 5 = 250px */
}

/* 推荐全局设置 */

*::after {

/* ==================== 问题1：外边距合并 ==================== */

/* 垂直方向相邻的块级元素，外边距会合并（取最大值） */
.box1 {
  margin-bottom: 20px;
}

.box2 {
  margin-top: 30px;
}
/* 实际间距 = max(20px, 30px) = 30px */

/* 解决方案：使用padding代替margin */
.box1 {
  margin-bottom: 0;
  padding-bottom: 20px;
}

.box2 {
  margin-top: 0;
}

/* ==================== 问题2：父子元素外边距塌陷 ==================== */
.parent {
  width: 300px;
  height: 200px;
  background-color: #f0f0f0;
}

.child {
  margin-top: 50px;  /* 这会让父元素一起下移 */
}

/* 解决方案1：给父元素添加padding */
.parent {
  padding-top: 50px;
}

.child {
  margin-top: 0;
}

/* 解决方案2：给父元素添加overflow */
.parent {
  overflow: hidden;
}

.child {
  margin-top: 50px;
}

/* 解决方案3：使用BFC */
.parent {
  display: flow-root;
}

.container {

  width: 300px;
  padding: 20px;
  border: 5px solid #ddd;

CSS属性值详解 ​

display 属性值详解 ​
值值的具体含义元素行为特点常见应用场景
block元素以块级方式显示，独占一行，前后有换行可设置 width/height，支持所有 margin/padding，宽度默认填满父元素div, p, h1-h6, form, header, section, article
inline元素以行内方式显示，与其他元素在同一行排列不可设置 width/height，垂直方向的 margin/padding 无效，宽高由内容决定span, a, label, strong, em, code
inline-block元素既像 inline 一行显示，又像 block 可设尺寸可设置 width/height，垂直 margin/padding 有效，但不独占一行按钮组、标签、图标+文字组合、内联卡片
flex元素成为弹性容器，子元素按弹性盒模型排列子元素默认横向排列，可通过 flex 属性控制伸缩导航栏、工具栏、卡片内部布局、分栏布局
inline-flex与 flex 相同，但容器本身不独占一行容器像 inline 一样与其他元素同行排列小型弹性组件、行内工具栏
grid元素成为网格容器，子元素按二维网格排列可同时定义行和列，子元素可跨越多个网格页面整体布局、卡片网格、仪表盘布局
inline-grid与 grid 相同，但容器本身不独占一行容器像 inline 一样与其他元素同行排列小型网格组件、行内网格展示
none元素完全隐藏，从页面中消失元素不占据任何空间，如同不存在隐藏弹窗、条件显示内容、切换显示状态
contents容器元素本身不显示，但子元素正常显示容器从渲染树中消失，子元素如同容器的直接子元素CSS Grid 中的辅助包装器
table元素表现为表格（类似 <table>）子元素应使用 table-row, table-cell 等值模拟表格布局（不推荐）
table-cell元素表现为表格单元格可使用 vertical-align，多个元素同行等高模拟表格单元格、实现等高布局
list-item元素表现为列表项类似 <li> 的效果，前面有列表标记自定义列表样式
position 属性值详解 ​
值值的具体含义定位参照对象元素行为特点
static元素按照正常文档流排列，不做任何特殊定位无特殊参照，按正常流排列元素在页面中的自然位置，top/right/bottom/left 属性无效
relative相对于元素在正常流中的原始位置进行偏移相对于自己原来应该在的位置元素仍占据原位置空间，偏移后的位置可能与其他元素重叠
absolute元素脱离正常文档流，相对于最近的定位祖先元素定位相对于最近的 position 不为 static 的祖先元素不占据原位置空间，完全脱离文档流，可与其他元素重叠
fixed元素固定在浏览器窗口的某个位置，不随页面滚动而移动相对于浏览器窗口（视口）不占据原位置空间，始终固定在视口中的同一位置
sticky元素在正常流中排列，但在滚动到特定位置时"粘"住在达到阈值前相对自身定位，达到后相对于滚动容器定位兼具 relative 和 fixed 的特性，适合吸顶效果

定位配合属性：

  • top：距离参照对象顶部的偏移量（负值向上）
  • right：距离参照对象右侧的偏移量（负值向左）
  • bottom：距离参照对象底部的偏移量（负值向上）
  • left：距离参照对象左侧的偏移量（负值向右）
  • z-index：设置层叠顺序，数值越大越靠上，只对定位元素生效
flex-direction 属性值详解 ​
值值的具体含义主轴方向子元素排列效果
row（默认）主轴为水平方向，子元素从左到右排列→ 水平向右[1][2][3][4] 一行从左到右
row-reverse主轴为水平方向，子元素从右到左排列← 水平向左[4][3][2][1] 一行从右到左
column主轴为垂直方向，子元素从上到下排列↓ 垂直向下[1]→[2]→[3] 一列从上到下
column-reverse主轴为垂直方向，子元素从下到上排列↑ 垂直向上[3]→[2]→[1] 一列从下到上

使用场景：

  • row：大多数水平布局场景（导航栏、工具栏、按钮组）
  • column：垂直布局场景（侧边栏菜单、卡片内垂直排列、表单字段）
  • row-reverse：需要反向顺序的水平布局（RTL语言支持）
  • column-reverse：需要反向顺序的垂直布局（倒序显示的消息列表）
flex-wrap 属性值详解 ​
值值的具体含义换行行为子元素尺寸变化
nowrap（默认）所有子元素在一行内显示，不换行强制单行显示子元素可能被压缩以适应容器宽度
wrap子元素换行显示，从上到下、从左到右排列多行显示，第一行在上方子元素保持正常宽度，容器放不下时换行
wrap-reverse子元素换行显示，从下到上、从左到右排列多行显示，第一行在下方子元素保持正常宽度，排列方向相反

使用场景：

  • nowrap：横向导航栏、工具栏（确保所有项目在一行）
  • wrap：标签列表、卡片网格、图片列表（允许换行）
  • wrap-reverse：特殊布局需求（较少使用）

flex-end所有子元素在交叉轴终点对齐子元素紧贴交叉轴终点，起点方向可能留有空白底部对齐（主轴为row时）
center所有子元素在交叉轴方向上居中对齐子元素整体在交叉轴方向居中，两侧留有相等空白垂直居中（主轴为row时）
baseline所有子元素按文字基线对齐按照元素内文字的基线对齐，适用于有文字的元素文字基线对齐，适合混合文字和图标

  • align-items：控制单行内子元素的对齐方式
  • align-content：控制多行之间的对齐方式（只在有多行时生效）
flex-grow / flex-shrink / flex-basis 详解 ​
属性默认值值的具体含义使用效果

flex-shrink1定义子元素的缩小比例，0表示不缩小，数值越大被压缩得越多容器空间不足时，值为0的元素保持原尺寸，值为1的元素会被压缩
flex-basisauto定义子元素在主轴方向上的初始大小，设置分配空间前的基础尺寸设置为200px时，元素先占200px，剩余空间再按flex-grow分配
flex0 1 auto简写属性，依次是 grow、shrink、basisflex: 1 = flex-grow: 1; flex-shrink: 1; flex-basis: 0%

opacity vs visibility 对比 ​
属性值值的具体含义是否占据空间是否响应事件是否可动画化
opacity0-1透明度，0完全透明，1完全不透明是（透明时仍占据空间）是（透明时仍可点击）是（可平滑过渡）
visibilityvisible/hidden可见性，hidden时隐藏是（隐藏时仍占据空间）否（hidden时不响应事件）否（只有显示/隐藏两种状态）

选择建议：

  • 需要平滑淡入淡出效果 → 使用 opacity
  • 需要隐藏但保留布局空间 → 使用 visibility: hidden
  • 需要隐藏且释放布局空间 → 使用 display: none

translate（平移）

值详细说明示例

animation 动画属性详解 ​
属性说明常用值
@keyframes定义动画关键帧@keyframes name { from/to } 或 { 0% / 50% / 100% }
animation-name动画名称自定义名称
animation-duration动画时长0.3s, 1s, 2.5s
animation-timing-function时间曲线同 transition-timing-function
animation-delay延迟时间0.1s, 1s
animation-iteration-count重复次数1, 2, infinite（无限）
animation-direction动画方向normal, reverse, alternate, alternate-reverse
animation-fill-mode填充模式none, forwards, backwards, both
animation-play-state播放状态running, paused
animation简写属性name duration timing-function delay iteration-count direction fill-mode

animation-direction 详解：

值详细说明
normal（默认）每次循环都正常播放
reverse每次循环都反向播放
alternate奇数次正常，偶数次反向（来回循环）
alternate-reverse奇数次反向，偶数次正常

animation-fill-mode 详解：

值详细说明
none（默认）动画前后不应用任何样式
forwards动画结束后保持最后一帧样式
backwards动画等待期间应用第一帧样式
both同时应用 forwards 和 backwards
