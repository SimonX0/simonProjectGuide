## 中级面试题

### 容器化与编排

核心知识点：
- Docker核心原理
- 镜像优化最佳实践
- Kubernetes架构
- Pod、Service、Ingress
- Deployment与StatefulSet
- ConfigMap与Secret

### CI/CD基础

核心知识点：
- CI/CD核心概念
- Jenkins Pipeline
- GitLab CI/CD
- GitHub Actions
- 构建优化策略
- 自动化测试集成

### 监控与日志

核心知识点：
- 监控体系设计
- Prometheus与Grafana
- 日志收集与分析（ELK/Loki）
- 告警策略设计
- APM与链路追踪
- 可观测性三大支柱

## 高级面试题

### 云原生架构

核心知识点：
- 云原生12要素应用
- 微服务架构设计
- 服务网格（Istio/Linkerd）
- 云平台实践（AWS/阿里云）
- Serverless架构
- 混合云与多云管理

### DevSecOps与安全

核心知识点：
- DevSecOps理念
- 容器安全最佳实践
- 镜像扫描与签名
- 密钥管理
- 合规性与审计
- 安全左移实践

### 服务网格与GitOps

核心知识点：
- 服务网格架构原理
- Istio核心组件（Pilot、Citadel、Mixer）
- 流量管理与灰度发布
- GitOps核心理念
- ArgoCD与Flux
- 声明式基础设施

## 项目实战

- **项目三**：全链路监控体系（从0到1搭建）
- **项目四**：GitOps自动化部署平台

核心知识点：
- 真实项目架构设计
- 技术选型决策
- 性能优化实战
- 成本控制方案
- 故障排查案例
- 面试STAR法则

1. **理论结合实践**：每个知识点都要动手验证
2. **关注官方文档**：Kubernetes等官方文档最权威

4. **考取认证**：CKA、CKS等认证含金量高
