## 📘 中级面试题（1-3年经验）

### 学习重点

中级面试题主要考察：
- **扎实的SQL基础**（复杂查询、子查询、窗口函数）
- **索引设计能力**（索引类型、复合索引、优化）
- **事务与锁机制**（隔离级别、锁类型）
- **MySQL基础优化**（慢查询、执行计划）
- **Redis基础应用**（数据类型、缓存策略）

### 面试题列表

#### MySQL基础与优化
- MySQL 8.0新特性有哪些？
- 索引的数据结构及优缺点
- 聚簇索引与非聚簇索引区别
- 复合索引的最左前缀原则
- 哪些场景索引会失效？
- 如何分析慢查询？
- EXPLAIN执行计划各字段含义
- MySQL执行查询的流程
- B+树为什么比B树更适合数据库索引？

#### 事务与锁机制
- MySQL事务隔离级别及实现原理
- MVCC（多版本并发控制）原理
- RR级别下如何解决幻读？
- 行锁、表锁、间隙锁区别
- 死锁产生原因及解决方案
- 乐观锁与悲观锁如何实现？
- 当前读与快照读区别

#### 主从复制与高可用
- MySQL主从复制原理（binlog三种格式）
- 主从延迟如何解决？
- 主从切换方案（MHA、Orchestrator）
- GTID是什么？有什么优势？
- 半同步复制原理

#### Redis基础与缓存
- Redis数据结构及应用场景
- Redis为什么快？（IO多路复用、单线程模型）
- 缓存穿透、击穿、雪崩区别及解决方案
- 如何保证缓存与数据库一致性？
- Redis持久化方式（RDB、AOF）
- Redis主从、哨兵、集群模式
- 布隆过滤器原理及应用

## 📕 高级面试题（3-5年经验）

### 学习重点

高级面试题主要考察：
- **数据库架构设计**（分库分表、读写分离）
- **分布式事务**（2PC、TCC、Saga、本地消息表）
- **国产分布式数据库**（OceanBase、TiDB、达梦）
- **NoSQL与向量数据库**（MongoDB、Elasticsearch、Milvus）
- **数据库迁移与备份**（冷备、热备、迁移方案）

### 面试题列表

#### 分库分表架构
- 什么情况下需要分库分表？
- 垂直分库与水平分库区别
- 分表策略（范围、Hash、一致性Hash）
- 分表后如何处理分页查询？
- 分表后如何保证全局唯一ID？（雪花算法）
- 分表后如何扩容？
- 分库分表带来的问题及解决方案

#### 分布式事务
- CAP理论与BASE理论
- 分布式事务解决方案对比（2PC、TCC、Saga、本地消息表）
- Seata AT模式与TCC模式区别
- 如何实现分布式事务的最终一致性？
- Saga事务的补偿机制如何设计？

#### 国产分布式数据库
- OceanBase的Paxos协议及三地五中心架构
- TiDB的HTAP架构及TiKV存储引擎
- 达梦数据库与Oracle的兼容性
- openGauss的AI4DB特性
- 国产数据库迁移方案及注意事项

#### NoSQL与向量数据库
- MongoDB的副本集与分片集群
- Elasticsearch的倒排索引原理
- Milvus向量索引类型（HNSW、IVF）
- 向量数据库在RAG中的应用
- 时序数据库（InfluxDB、TDengine）应用场景

#### 数据库迁移与备份
- MySQL冷备份与热备份方案
- 如何实现零停机时间迁移？
- 国产数据库迁移的兼容性问题
- 大数据量迁移的最佳实践

## 📚 学习建议

1. **理论结合实践**：在本地搭建环境验证
2. **对比学习**：MySQL vs PostgreSQL vs 国产数据库
3. **关注趋势**：国产数据库是信创重点
4. **项目经验**：准备1-2个数据库优化案例
5. **工具使用**：熟悉EXPLAIN、慢查询日志、监控工具
