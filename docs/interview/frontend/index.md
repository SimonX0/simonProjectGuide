## 中级面试题

### ：JavaScript基础面试题

核心知识点：
- 数据类型与类型判断
- 闭包与作用域
- this与执行上下文
- 原型与继承
- 异步编程
- 事件循环

### ：Vue3核心面试题

核心知识点：
- 响应式原理
- ref与reactive
- 组合式API
- computed与watch
- 生命周期

### ：组件开发面试题

核心知识点：
- 组件基础
- Props与Emit
- 插槽Slots
- 组件通信
- 组件生命周期

### ：路由与状态管理面试题

核心知识点：
- Vue Router核心概念
- 路由模式与传参
- 路由守卫
- Pinia状态管理
- Store设计

## 高级面试题

### ：性能优化面试题

核心知识点：
- 渲染性能优化
- 虚拟列表
- 内存优化
- 防抖与节流
- 加载性能优化

### ：工程化面试题

核心知识点：
- Vite构建工具
- 模块化与包管理
- 代码规范
- CI/CD
- 测试

### ：架构设计面试题

核心知识点：
- 设计模式
- MVVM模式
- 组件设计原则
- 状态管理架构
- 微前端架构

### ：前端安全面试题

核心知识点：
- XSS攻击与防御
- CSRF攻击与防御
- 其他安全威胁
- 安全最佳实践

## 学习建议

1. **按顺序学习**：从中级开始，打牢基础
2. **理解原理**：不仅要记住答案，更要理解背后的原理
3. **动手实践**：将理论知识转化为代码实践
4. **项目结合**：用实际项目经验佐证你的理解
