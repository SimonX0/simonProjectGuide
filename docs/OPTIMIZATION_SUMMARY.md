## 十、使用指南

### 学习者使用

1. **查看首页** - 了解完整学习路径
2. **选择路线** - 根据基础选择合适路径
3. **追踪进度** - 自动保存学习进度
4. **按计划学习** - 参考时间规划

### 维护者使用

#### 添加新章节
```markdown
# 在对应的index.md中添加章节信息
**[第X章：章节标题 →](chapter-x)**
- 内容要点1
- 内容要点2
```

#### 修改样式
```css
/* 编辑 custom.css */
.learning-progress-card {
  /* 修改卡片样式 */
}
```

#### 更新进度追踪
```javascript
// 在index.md的script中添加新章节
const chapters = [
  { id: 'chapter-new', title: '新章节' },
  // ...
];
```
