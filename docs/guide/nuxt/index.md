## 为什么学习 Nuxt？

### Nuxt 的核心优势

- ✅ **基于 Vue 3**：熟悉的 Vue 3 语法和 Composition API
- ✅ **服务端渲染（SSR）**：SEO 友好，首屏加载快
- ✅ **自动路由**：基于文件系统的自动路由生成
- ✅ **自动导入**：组件、组合式函数自动导入
- ✅ **服务端引擎（Nitro）**：强大的服务端能力
- ✅ **类型安全**：原生 TypeScript 支持
- ✅ **多平台部署**：支持多种部署平台

### 技术栈

本教程使用以下技术栈：

- **Nuxt 3+ / 4** - 最新版 Nuxt 框架
- **Vue 3** - UI 框架
- **TypeScript** - 类型安全的 JavaScript 超集
- **Pinia** - 状态管理方案
- **Nitro** - 服务端引擎
- **Prisma** - ORM 数据库工具

## 学习目标

完成本教程后，你将能够：

- ✅ 扎实掌握 Nuxt 3/4 核心概念

1. **先掌握 Vue 3**：确保有扎实的 Vue 3 和 Composition API 基础
2. **按顺序学习**：Nuxt 概念较多，建议循序渐进
3. **动手实践**：每个特性都要亲自尝试
4. **对比学习**：理解 SSR、SSG、ISR 的区别和适用场景
5. **使用 AI 辅助**：善用 Claude、ChatGPT 等工具提升效率
6. **部署实践**：使用 Vercel、Cloudflare 等平台部署项目
