## 为什么学习 React？

### React 的核心优势

- ✅ **组件化开发**：可复用的 UI 组件，提升开发效率
- ✅ **虚拟 DOM**：高效的渲染机制，性能优异
- ✅ **生态强大**：丰富的第三方库和工具
- ✅ **就业市场需求大**：大量企业采用 React 技术栈
- ✅ **Facebook 官方支持**：持续更新，React 19 带来更多新特性

### 技术栈

本教程使用以下技术栈：

- **React 18+ / 19** - 最新版 React 核心框架
- **React Router 6+** - React 官方路由管理器
- **Zustand / Jotai** - 现代状态管理方案
- **TanStack Query** - 强大的数据同步库
- **Vite** - 下一代前端构建工具
- **TypeScript** - 类型安全的 JavaScript 超集
- **Vitest** - 快速的单元测试框架
