## 为什么学习 Next.js？

### Next.js 的核心优势

- ✅ **服务端渲染（SSR）**：SEO 友好，首屏加载快
- ✅ **App Router**：最新的路由系统，功能强大
- ✅ **Server Components**：减少客户端 JS 体积
- ✅ **文件系统路由**：基于文件系统的自动路由
- ✅ **API Routes**：内置 API，可构建全栈应用
- ✅ **Vercel 优化**：官方部署平台，零配置部署
- ✅ **生态强大**：丰富的集成和插件

### 技术栈

本教程使用以下技术栈：

- **Next.js 14+ / 15** - 最新版 Next.js 框架
- **React 18+** - UI 框架
- **TypeScript** - 类型安全的 JavaScript 超集
- **Tailwind CSS** - 实用优先的 CSS 框架
- **Vercel** - 官方部署平台
- **Zustand** - 状态管理方案

## 学习目标

完成本教程后，你将能够：

- ✅ 扎实掌握 Next.js 14/15 核心概念
- ✅ 熟练使用 App Router 和 Server Components
- ✅ 理解 SSR、SSG、ISR 等渲染模式
- ✅ 掌握 Server Actions 和数据获取
- ✅ 能够构建全栈应用
